АНАЛІЗАТОР ТЕКСТУ (Go)

Опис:
Програма аналізує текст, введений користувачем:
- Підраховує кількість слів у тексті
- Знаходить кількість входжень заданого слова (незалежно від регістру)
- Визначає найдовше слово та його довжину (у символах)
- Знаходить перше слово, що починається на задану літеру
- Дозволяє повторити аналіз з новим текстом

Використання:
1. Запустіть програму командою:
   go run main.go
2. Дотримуйтесь інструкцій у консолі:
   - Введіть текст для аналізу
   - Введіть слово для пошуку
   - Введіть літеру для пошуку першого слова
   - Виберіть, чи бажаєте проаналізувати інший текст

Вимоги:
- Go 1.18 або новіше

Приклад роботи:
=== АНАЛІЗАТОР ТЕКСТУ ===
Введіть текст для аналізу:
Go is an open source programming language. Go makes it easy to build simple, reliable, and efficient software.
Введіть слово або літеру для пошуку: go
Слово або літера "go" зустрічається 2 раз(и).
У тексті всього 15 слів.
Найдовше слово тексту: programming (11 символів)
Введіть літеру для пошуку першого слова:
p
Перше слово, що починається на "p": programming
Хочете проаналізувати інший текст? (yes/no)
